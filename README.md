# NurPath

NurPath is a gamified Islamic habit-tracking application designed to help users build consistent routines around their sacred duties, sunnah prayers, and additional virtuous acts.

## Features

- **Gamified Tracking**: Earn XP for completing daily Salah, Adhkar, and optional Sunnahs.
- **Prayer Packages**: Automated checklists for Pre-Salah (Miswak, Wudhu, Tahiyyatul Masjid) and Post-Salah (Tasbeeh Fatimi, Adhkar, Surah) nested under the main daily prayers.
- **Ramadan Tracker**: Dedicated 30-day Ramadan calendar tracker to earn XP for daily fasts.
- **Dynamic "Today's Focus"**: Highlights the upcoming or current prayer to guide users to their priority action.
- **Routine Builder**: Build a personalized daily routine containing Main, Sunnah, and Community quests. 

## Technology Stack

- **Frontend**: React (Vite + TypeScript)
- **Styling**: TailwindCSS
- **Backend & Auth**: Supabase (PostgreSQL, Supabase Auth)
- **Deployment**: Vercel
- **Icons**: Lucide React

## Key Scripts

- `npm run dev` - Starts the local development server.
- `npm run build` - Builds for production.
